# 🛋️ Zenvora Living – End-to-End E-Commerce Testing Project

## 📌 Project Overview
Zenvora Living is a full-stack furniture e-commerce application.  
This project demonstrates complete software testing coverage including Manual, API, Automation, Performance, and CI/CD testing.

---

## 🏗️ Technology Stack

**Frontend:**  
HTML, CSS, JavaScript  

**Backend:**  
Spring Boot (REST APIs)  

**Database:**  
MySQL  

**Build Tool:**  
Maven  

**CI/CD:**  
Jenkins  

**Testing Tools:**  
Postman, Selenium WebDriver, JMeter  

---

## 🧪 Testing Coverage

### 🔹 Manual Testing
- Functional Testing  
- Negative Testing  
- Test Case Design  
- Defect Reporting  
- End-to-End Scenario Validation  

### 🔹 API Testing (Postman)
- User Signup API  
- User Login API  
- Product APIs  
- Cart APIs  
- Checkout APIs  
- Order APIs  

Validated:
- Status Codes  
- Request & Response Body  
- Error Handling  
- Data Validation  

### 🔹 Automation Testing (Selenium)
- Login Automation  
- Add to Cart  
- Checkout Flow  
- Wishlist  
- Search Functionality  
- Logout  

Framework:
- Page Object Model (POM)  
- TestNG  
- Maven Integration  

### 🔹 Performance Testing (JMeter)
Full E-commerce Flow:

Login → Products → Add to Cart → View Cart → Checkout  

Includes:
- CSV Data-Driven Testing  
- Assertions  
- Load Testing Scenarios  

### 🔹 CI/CD Integration (Jenkins)
- GitHub Integration  
- Automated Build Execution  
- Maven Build  
- Continuous Testing Pipeline
